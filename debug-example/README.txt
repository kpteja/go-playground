dlv debug                   - Start debugging
(dlv) continue              - Continue until next breakpoint
(dlv) breakpoints           - List all breakpoints
(dlv) break main.go:21      - Add a breakpoint at line no 21 in main.go file
(dlv) b main.go: 19         - b is alias for break
(dlv) clear 1               - Clear breakpoint 1
(dlv) clearall              - Clear all breakpoints
(dlv) next                  - Go to next step
(dlv) step                  - Step into a function
(dlv) stepout               - Step out of a function
(dlv) stack                 - Displays stack trace
(dlv) q                     - Quit debugging
